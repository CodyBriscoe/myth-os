# TheDrummer Model Notes

Use these notes when pairing the starter kit with a TheDrummer-style RP model.

- Best for: long-form character interaction, atmospheric narration, slow-burn social pressure, mystery, and consequence-heavy RP.
- Prompting style: give firm role boundaries, active NPC motives, and concrete scene state.
- Context hygiene: keep lorebook entries compact. Summaries should preserve decisions and relationships over prose.
- Length control: use `max_tokens` first, then author note. If responses become too ornate, lower temperature and add "favor concrete language" to author notes.
- Continuity: repeat critical facts through lorebook, memory, or summary instead of relying on old chat context.
- GM cards: use the narrator card when user wants guided play. Use character cards when user wants focused relationship scenes.
- Starter Campaign v1: use Ashwake Crossing when you want to test immediately without filling templates.
- Sampler variants: start Balanced, use Tight for clue work/combat, use Cinematic for reveals and emotional scenes.
- Image generation: not included in this kit.

Recommended starting sampler lives at `config/sillytavern_thedrummer_sampler_preset.json`.
Variant sampler values live at `config/sillytavern_thedrummer_sampler_variants.json`.
