# Summary And Memory Prompts

## Rolling Summary Prompt

Summarize the roleplay so far for future continuity. Preserve:

- Current scene, location, time, and immediate stakes.
- `{{user}}` goals, choices, injuries, resources, promises, and secrets.
- Important NPC names, motives, attitudes toward `{{user}}`, and last known status.
- Factions, threats, mysteries, clues, debts, and unresolved hooks.
- Tone, boundaries, and any user preferences established in chat.

Write concise bullet points. Do not add new facts.

## Character Memory Prompt

Update durable memory for recurring characters. For each important character, record:

- Name and role.
- What they want.
- What they believe about `{{user}}`.
- Trust level and emotional temperature.
- Secrets known, promises made, conflicts, injuries, or debts.
- What they are likely to do next if left alone.

## Scene Handoff Prompt

Create a compact handoff for continuing later:

- Last visible moment.
- What `{{user}}` was about to decide or do.
- Active NPCs present and their posture.
- Environmental hazards or opportunities.
- Hidden pressures currently moving offscreen.

## Memory Cleanup Prompt

Condense old memory. Keep only facts that affect future play. Remove duplicate wording, obsolete scene detail, and guesses that were disproven.
